4. HTTP의 약자는?
    - hyperText Transfer Protocol

5. 쿠키의 규칙?
    - 쿠키는 도메인 1개만 한정한다
    - 쿠키는 컴퓨터에 자동으로 저장된다

    - X 쿠키는 반드시 동의를 받은 후 받을 수 있다.
        (대부분의 웹사이트에서는 자동으로 수집한다)

    - X

8. 이미지손상을 최소화 하면서 용량을 줄일 수 있는 알고리즘?
